# Koala Conservation & Physical Health Analysis

## 📌 Project Overview
This project explores the physiological characteristics of koalas across Queensland (QLD) and Victoria (VIC). Using statistical modeling, I analyzed how habitat and environmental factors influence physical growth markers like head length, foot size, and body circumference.

## 🛠️ Data Science & Statistical Workflow
* **Data Source:** Processed research data provided in Excel format (`.xlsx`).
* **Exploratory Data Analysis (EDA):** Analyzed physical distributions using Boxplots and Histograms to identify regional outliers.
* **Statistical Modeling:** * Implemented **Correlation Heatmaps** to visualize the relationship between body segments.
    * Performed **Hypothesis Testing** (using Scipy) to verify significant differences between regional populations.
* **Advanced Visuals:** Used Seaborn for multi-variate analysis of age vs. physical development.

## 💡 Key Insights
* **Growth Indicators:** Identified that head length is a highly reliable predictor of total body length, providing a non-invasive way to estimate age and health in the field.
* **Environmental Impact:** Significant physiological variations were found between QLD and VIC populations, highlighting the need for localized conservation strategies.
* **Belly & Chest Correlation:** Found a strong positive correlation between nutritional markers and habitat quality.
